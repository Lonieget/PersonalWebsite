關於我

鄭宇翔

全端開發者 & 軟體工程師

在大學期間，我參與了以 Unreal Engine 製作的 VR 解謎遊戲專題。透過設計藍圖（Blueprint）與實作互動邏輯，我深刻體會到了程式如何為虛擬世界注入靈魂，這也確立了我朝軟體工程師發展的決心。

退伍後，我參加了產業新尖兵計畫的万十科技全端工程師培訓。在這裡，我完成了「動態問卷系統」與「POS 系統」專案，掌握了 Angular 的前端開發與 Java、C# 的後端思維，並熟悉了 API 串接與 Git 團隊協作流程。

我重視程式碼的可維護性，喜歡在透徹理解系統架構後進行優化。未來，我期許自己能成為一位獨當一面的全端開發者，為團隊帶來實質且穩定的技術貢獻。
